# YC — 스타트업 아이디어 얻고 평가하는 법 (Jared Friedman, Startup School)

## 1. 핵심 요약
YC 파트너 Jared Friedman의 강연. 좋은 아이디어를 *고르는 것*이 성공 확률을 높인다는 전제. **해법이 아니라 문제와 사랑에 빠지고**, 결국 **창업자-시장 적합성**이 가장 중요하다는 게 핵심. 근거: YC 시총 top 100 기업의 아이디어 출처 정량분석 + Paul Graham 에세이 'How to Get Startup Ideas' + 피벗 코칭 경험 + 탈락 지원서 수천 건 분석. 구성: ①흔한 실수 ②평가 10문항 ③생성 7레시피.

## 2. Key Takeaways

### ① 가장 흔한 실수 4
- **진짜 문제 미해결(SISP, solution in search of a problem):** "AI 멋진데 어디 쓰지?"식 접근. → 문제에서 출발. 단 너무 추상적(세계빈곤)이면 안 되고 *구체적·해결가능*해야.
- **타르 구덩이(tar pit) 아이디어:** 모두가 겪는 흔한 문제라 쉬워 보이나 *구조적 이유*로 아무도 못 푼 것(예: 친구 약속잡기 앱, 20년째 YC 지원). → 구글링 + 과거 도전자 인터뷰로 '어려운 지점' 파악.
- **첫 아이디어에 무작정 뛰어듦** (사업성 검토 생략).
- **완벽한 아이디어 기다리다 시작 못 함.** 정답은 중간. 아이디어 = 변형될 *출발점*.

### ② 아이디어 평가 — 10문항
1. **창업자-시장 적합성(founder-market fit)** — 가장 중요. *우리 팀*이 적임자인가. 아이디어 탐색을 "우리 팀에 좋은 아이디어 찾기"로 재정의. (PlanGrid)
2. **시장 크기** — 지금 크거나 / 작지만 급성장(2012 Coinbase).
3. **문제의 절박함** — 대안이 *문자 그대로 없음*이 최고(Brex 스타트업용 법인카드).
4. **경쟁자** — 있는 게 정상. 강한 기존 경쟁엔 *새 통찰* 필요.
5. **본인/지인이 진짜 원하나.**
6. **최근에야 가능/필요?** — 신기술·규제변화·새 문제가 기회 창출(Checker, 긱이코노미 배경조사 API).
7. **좋은 프록시(proxy)** — 유사하나 직접 경쟁 아닌 대기업(Rappi ← DoorDash 중남미).
8. **수년간 할 만한가** — 함정. 열정은 나중에 생김. 최고 아이디어는 지루한 영역에 많음.
9. **확장 가능성** — 순수 SW=O / 서비스·에이전시(고숙련 인력 의존)=주의.
10. **좋은 아이디어 공간(idea space)** — Dalton 개념. 영역별 적중률 천차만별: 핀테크 인프라·버티컬 SaaS↑ vs 소비자 하드웨어·소셜·애드테크↓. 비옥한 공간이면 초기 아이디어가 틀려도 인접 아이디어로 표류 가능(Fivetran).

### ③ "나빠 보이지만 사실 좋은" 3가지 (남들이 피해서 기회로 남음)
- **시작이 어려운 것:** Stripe(은행 딜·결제 인프라 학습이 다 겁먹게 함 = schlep blindness, PG 에세이).
- **지루한 영역:** Gusto(급여 SW). 어차피 6~12개월 후 일상 업무는 동일 → 초기 '재미'는 실제 즐거움과 무상관.
- **경쟁자 있는 것:** Dropbox(20번째였으나 *아무도 안 씀* → "다들 UI가 구리다" + OS 통합 자동동기화 통찰).

### ④ 아이디어 생성
- **최선은 유기적 발견** (top 100의 70%+). 장기 셋업: 가치있는 분야 전문가 되기(스타트업 취업) / 프로그래머는 흥미로운 것 만들기(Replit).
- **즉시 만드는 7레시피(좋은 순):** (1) 팀이 잘하는 것 → 자동 founder-market fit(Rezi) (2) 직접 겪은 문제, 남보다 잘 보이는 위치(Vetcove) (3) 내가 원하는 것(DoorDash) ← 가장 위험, "왜 아직 없지?" 자문 (4) 최근 바뀐 것(Gather.town/코로나) + 최근 성공기업의 변형(Nuvocargo=중남미 Flexport) (5) 사람들에게 문제 묻기 — 비옥한 공간 먼저 고르고 고객+창업자 둘 다 인터뷰(AtoB, 트럭 휴게소 방문) (6) 망가진 거대 산업 (7) [보너스] 아이디어 있는 공동창업자 찾기.
- **마무리:** 확신의 유일한 방법 = **그냥 출시해서 확인**.

## 3. 💡 사업 아이디어 (한화 적용 관점)
- **10문항 = 한화 신사업 심사 체크리스트:** 사내 신사업 제안을 이 10개 질문으로 스크리닝하면 'AI 멋지니까' 식 SISP 제안을 거를 수 있음.
- **'창업자-시장 적합성' → '계열사-시장 적합성':** 한화가 *기존 자산*(보험/증권/자산운용/에너지/방산/리조트)을 레버리지하는 신사업은 자동 적합성 확보. 무관한 영역 진출은 경계.
- **대기업의 역설적 강점 = "시작이 어렵고/지루하고/규제 심한" 영역:** schlep(은행 딜·규제 대응·인프라)이 스타트업을 겁주는 곳이야말로 *자본·라이선스·관계*를 가진 한화가 이기는 자리. (앞서 본 [[meta-arena-prediction-market]]의 도박규제, [[uipath-maestro-case-agentic]]의 금융 컴플라이언스가 바로 그런 '진입장벽=해자' 사례.)
- **'최근에야 가능/필요' 렌즈:** STO 제도화, 탄소시장 개설, LLM 등 *규제·기술 변화*가 한화 금융·에너지 계열의 신사업 창구.
- **'좋은 아이디어 공간' 선택:** 코퍼레이트 벤처링도 적중률 높은 공간(핀테크 인프라·버티컬 SaaS)부터.
- **AtoB식 현장조사:** 책상이 아니라 *현장(계열사 영업점·고객)*에 가서 고객+업계 창업자 인터뷰 → 진짜 문제 발굴.
