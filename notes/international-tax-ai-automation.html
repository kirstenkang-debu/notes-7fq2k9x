◆핵심 파이프라인 — 4단계 & 단계별 기술 스택

➜
STEP 1

자료 취합 (Ingest)

  • HCON 연결데이터 API 수취
  • 중간연결정산표 엑셀 수집(메일/공유폴더)
  • 법인·기간 메타데이터 태깅
필요 기술 HCON API/DB 커넥터 IMAP/Graph 메일 수집 openpyxl · 파일 스토리지 문서 버전·계보 관리
➜
STEP 2

데이터 추출 (Extract)

  • 이종 양식 → 표준 스키마 매핑
  • 셀 고정참조 X → 의미 기반 항목 인식
  • 신뢰도 낮으면 사람 검토 라우팅
필요 기술 (병목 핵심) LLM 구조화추출(Claude, structured output) Table/Layout 인식 계정과목 시맨틱 매핑 confidence score + HITL
➜
STEP 3

오류 검증 (Validate)

  • 정합성 룰(차대·합계·부호·통화)
  • HCON 대사(reconciliation)
  • 이상치·전기대비 급변 탐지
필요 기술 Rule Engine (검증규칙) Reconciliation 대사엔진 이상탐지(통계/ML) ETR·GloBE 임계치 체크
STEP 4

재검토·독촉 (Follow-up)

  • 미회신 법인 자동 탐지·집계
  • 오류사유 파악 → 재검토 메일 자동생성
  • 회신 트래킹 → 내역 확정
필요 기술 워크플로 오케스트레이션 LLM 메일 초안 + 승인후 발송 상태 대시보드·SLA 추적 감사로그(audit trail)
